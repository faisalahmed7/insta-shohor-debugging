How Does Java Script Work

Java Script is high level, dynamic and client side scripting language. Basically it runs on the side of users or clients end line by line by their web browsers. Because modern web browsers support JavaScript and JavaScript engine that runs the codes.At first the codes are taken to the engine as a parser. Then it checks any kinds of error line by line in the codes. Then parser makes a AST(Abstract Syntax Tree) if there are no error in the codes. So when the AST is created by the help of the parser, the JavaScript Engine like (v8, Chakra, JavaScript Core etc.) converts the code into byte code or machine codes. Here the JS engines act like Just In Time Compiler(JIT). So when the JS code is converted to the byte code or machine code, it is sent to the engine for execution and we get the output.
